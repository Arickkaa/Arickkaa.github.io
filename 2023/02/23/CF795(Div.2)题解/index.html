CodeCraft-22 and Codeforces Round #795 (Div. 2)

A. Beat The Odds

题意：

给定一个长度为 的整数序列，求最少删掉几个元素能让每两个相邻的元素相加为偶数

分析：

如果两个元素相加为偶数，只有当两个数同时为奇数或同时为偶数时才满足条件。因此对于整个序列，必须要每两个相邻的元素同奇或同偶。答案即序列中奇数个数和偶数个数的最小值。

B. Shoe Shuffling

题意：

给定一个长度为 的整数序列 并且 ，现在要改变序列的排列，新排列要求：

  • 每个元素不能在原来的位置
  • 每个元素的大小不能小于原来的元素

求任意一种满足条件的排列，如果排列不存在输出

分析：

如果有任何一个元素交换顺序后的值比原来的元素的值大，则必然会出现一个元素找不到比自己更大的元素，因此不满足条件。为满足要求，必须要使交换的元素和自身大小相同才行。

给出一种解决方案：

用双指针寻找相同元素的区间，将区间中的序号整体向右移动一位，然后将最后一个元素的序号移动到第一位。

E. Number of Groups

题意：

现在有 个红色或者蓝色的片段，每个片段覆盖区间 中的所有点。当两个片段颜色不同，并且共同覆盖了至少一个点时认为它们属于相同的组。求给出的片段最少属于几个组。

分析：

先将所有的片段的覆盖区间的开始点和结束点存储起来，从小到大遍历它们。

接下来通过维护两个 set 来进行并查集的合并操作：

  • 当遍历到的点是开始点：
    • 将这个点对应片段插入到对应的颜色的 set 中
    • 将另一个颜色的 set 中的所有点与这个点合并，并且删除另一个颜色的 set 中除最后一个点以外的所有点
  • 当遍历到的点是结束点：
    • 删除对应颜色的 set 中的这个片段

最后答案就是并查集中的组数。
