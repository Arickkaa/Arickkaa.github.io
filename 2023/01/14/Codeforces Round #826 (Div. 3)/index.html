D. Masha and a Beautiful Tree

解法一：

dfs，分治每两个子节点，通过找左右分支的最大值来判断是否需要交换，需要交换则结果加一

Solution(C++)

解法二：

可以看到这棵树是完全二叉树，并且是顺序的，因此满足下标为 $n$ 的节点的左右子节点的下标为 $2\times n$ 和 $2\times n + 1$。同时可以发现左右子的差值与这一层的高度满足 $\triangle_0=1,\triangle_i = 4\times i$，对每一层的进行判断是否满足条件和是否需要交换即可。

PS: 数组一定要开的够大，题目是 $3\times10^6$

（一开始开小了所以tle了两发，不然比赛中就对了）

E. Sending a Sequence Over the Network

题意：一个数组可以分成若干个长度任意的片段，现在发送这段数组，将它转换成另外一种格式，该格式为：在每一个片段的前后标明这个片段有几个元素。现在给定一个数组，判断是否符合发射数组的格式。

用dp去解决，状态转移方程有两个：

  • 首先是长度在前面： $dp[i - 1] == true\Rightarrow dp[i + b[i]] = true$
  • 还有是长度在后面： $dp[i - b[i] - 1] == true\Rightarrow dp[i] = true$

Solution(C++)
