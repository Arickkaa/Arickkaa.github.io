Codeforces Round 855 (Div. 3)

A - Is It a Cat?

题意：

给定一个字符串，判断这个字符串是否是meow的特定变形。

分析：

扫描这个字符串，看扫描完meow后是否字符串已经扫描到末尾，并且判断字符串是否包含四个字母（不区分大小写）。

B - Count the Number of Pairs

题意：

给定一个字符串，规定两个相同字母的不同大小写是一个组合，现在可以进行不超过k次操作，每个操作可以将一个大写字母变成小写或者将一个小写字母变成大写，求这个字符串操作后最多有多少个字母组合。

分析：

直接用 unordered_map 记录每个字母的出现次数，然后把原来就存在的组合数加上，再对多余的字母进行操作，再加上后就是答案。

C - Powering the Hero

题意：

现在有一堆牌，bonus牌有对应的权重，hero牌可以将前面的任何一张牌的权重加到军队中，并且舍弃这张bonus牌，求最大权重。

分析：

只需要用 priority_queue 将所有之前已经出现但未使用的 bonus牌的权重记录下来，遇到 hero牌的时候将权重最大的 bonus牌的权重加上。

D - Remove Two Letters

题意：

给定一个字符串，可以移除连续的两个字母，请问最多可以获得多少种不同的字符串。

分析：

对于每个字母，如果它的后两个下标对应的字母与它相同，则会有一次重复，需要减去。

E - Unforgivable Curse

题意：

给定两个字符串 s, t ，可以将每个字母和下标相距 k 或 k+1 位置的字母交换位置，判断 s 进行若干次交换后是否能变成 t 。

分析：

字符串中有一些不能交换的字母，就是 k 范围外没有字母，它们顺序不能变，其余可以交换的字母可以变成任意顺序。
