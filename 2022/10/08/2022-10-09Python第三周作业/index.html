Questions answered:

1.A Python list is considered to be an iterable. What does this mean?

If we want to access an element in a list,then use supply an index value to the subscript operator which represents the position of the element in the list.

So we can loop through the index values of a list using the length of the list in a range statement and we can also directly loop through the elements without using an index value

We can use keyword “for” to loop over all elements in a list.

2.What other Python data type that you have studied is also an iterable?

Dictionaries,string,tuple

3.Describe two ways that you can initialize a list of size N to have elements of the same value, e.g**. the value 0**

Replicate a list with one 0 element and multiply it by an integer.

aList = [0] * N

Using list comprehension, aList = [0 for n in range(N)]

4.What is a list comprehension and when would you use one? Give an example.

list comprehension can be used to create a list with specified or conditional values in one line

Zeros = [0 for n in range(10)]

evensTo20 = [n for n in range(21) if n % 2 == 0]

5.In Python, lists and tuples can perform many of the same tasks. Briefly explain their differences

the elements in tuples cannot be changed, but you can change a list

6.When would you use a list and when would you use a tuple? Give examples.

List and tuple are similar in so many aspects, but the most important difference is list can change its content after initialization but tuple can not . so , i will use list if I need to change the content after and use tuple if I need to declare an const array. Actually tuple is very useful to return multiple values from a function.

7.In languages like C, some applications require a linked list data structure to process dynamically-sized data. Why does Python not require linked lists?

Python doesn’t need to manage the memory, and list in python support to save and manipulate dynamically-sized data. list can do everything which the linked list data structure can do.

L2=L*2

print(L2)

10.What does the map() function do to a list? What is the return type from map()? Why?

the map() function is a function which takes a function and a list as two parameters, and returns a new list with the function applied to each element of the original list. I think map() function can be used to get a new list by changing the value of elements in the original list.

11.What does the reduce() function do to a list? What is the return type from reduce()?. Why?

the reduce() function is a function which takes a function and a list as two parameters, and returns an updated value as the function applied to the original list. I think reduce() function can be used to calculate something from the original list, for example, the sum

Are you more likely to use a functional programming approach in your future Python programming? Why?

Yes,Functional programming pays more attention to the results of program execution rather than the process of execution. It advocates using a number of simple execution units to make the calculation results gradually, and deduces complex operations layer by layer, rather than designing a complex execution process.
