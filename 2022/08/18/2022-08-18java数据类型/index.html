Java的几种数据类型

几种数据类型

java 是强类型语言，变量使用要严格符合规定，先定义再使用

数据类型名称 占用字节 默认值 封装器类
byte 1 0 Byte
short 2 0 Short
int 4 0 Integer
long 8 0.0l Long
float 4 0.0f Float
double 8 0 Double
char 2 \u0000 Character
boolean 2 false Boolean

类型转换

自动类型转换
低—————————————————————————————>高
byte、short、char->int->long->float->double

不需要任何操作

运算中，不同类型的数据先（自动）转化为同一类型然后进行运算

强制类型转换（高—>低）

注意点

  • 不能对布尔类型进行转换
  • 不能转换为不相干的类型
  • 把高容量转换到低容量的时候，强制转换
  • 转换的时候可能存在内存溢出，或者精度问题
