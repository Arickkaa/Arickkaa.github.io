变量、常量和作用域 & 包的导入

变量: 局部变量、类变量、实例变量

局部变量

局部变量：必须声明和初始化，只在声明的方法内有效

类变量

静态类变量(static)：可以在该类下的方法里直接使用

实例变量

实例变量：从属于对象；如果不自行初始化，这个类型取默认值（如下表）

实例变量的初始化值

元素类型 初始值
byte 0
short 0
int 0
long 0L
float 0.0f
double 0.0d
char “\u0000”
boolean false
对象引用 null

变量的命名

所有变量、方法、类名的要求：见名知意

类成员变量：驼峰原则：monthSalary

局部变量：驼峰原则

常量：大写字母和下划线：MAX_VALUE

类名：大驼峰原则：Man， GoodMan

方法名：驼峰原则run(), runRun()

常量

常量名一般使用大写字符

  • 类变量也叫静态变量，在类中以static关键字声明，但必须在方法之外
  • 无论一个类创建了多少个对象，类只有类变量的一份拷贝
  • 静态变量除了被声明为常量外很少使用，静态变量是声明为public/private、final和static类型的变量，初始化之后不可改变
  • 静态变量储存在静态存储区，经常被声明为常量，很少单独使用static声明变量
  • 静态变量在第一次被访问时创建，在程序结束时被销毁
  • 与实例变量具有相似的可见性，但为了对类的使用者可见，大多数静态变量声明为public类型
